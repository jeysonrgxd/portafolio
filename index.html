Habilidades

HTML5 es la última versión de HTML (lenguaje de marcado)

CSS3 es un lenguaje de marcado que se emplea para dar estilos a un sitio web

JavaScript es un lenguaje de programacion para dar a las webs interacción con el usuario.

Jquery libreria de javascript para facilitar el usu de dicho lenguaje para una mayor eficiencia.

Boostrap es un framework de css y javascript que nos ayuda a poder crear nuestra web mas rápidas.

Php lenguaje programacion de lado del servidor que ayudan a crear paginas dinamicas.

Wordpress es un poderosísimo cms que nos ayuda prácticamente en todo lo que tenga que ver con el desarrollo web.

Node.js es el lenguaje de programación JavaScript pero de lado del servidor.

Mysql es uno de los famosísimos gestores de base de datos relacional OpenSource.

MongoDb es una base de datos no relacional (No SQL) orientada a documentos JSON.

API es una interfaz o protocolo de comunicación entre diferentes partes de un programa.
